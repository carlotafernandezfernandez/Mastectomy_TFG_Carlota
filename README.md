# Overview

This repository contains the complete Unity project developed as part of my Bachelor's Thesis in Biomedical Engineering.

The project presents a standalone virtual reality prototype designed to support women preparing for mastectomy through progressive body familiarization and introductory postoperative self-care education.

The application combines immersive visualization, avatar personalization and natural hand-tracking interaction within a guided workflow specifically designed for Meta Quest 3.

Although developed as an academic proof of concept, the project explores the technical feasibility of using immersive virtual reality to support preoperative preparation and patient education.

---

# Main Features

- Customizable female avatars
  - Three body types
  - Four mastectomy configurations
  - Skin tone selection
  - Hairstyle customization
  - Hair colour selection
  - Eye colour customization

- Immersive virtual bathroom environment

- Real-time mirror visualization

- Guided interaction workflow

- Natural hand-tracking interaction

- Introductory postoperative self-care activities
  - Mirror observation
  - Shirt removal
  - Soap application
  - Water rinsing
  - Towel drying
  - Final postoperative recommendation
